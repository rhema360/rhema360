# guiones/

Guiones nuevos para Rhema360, cada uno con su simulación al final del
archivo (qué patrones cumple, señales a favor/en contra, estimación y
confianza). Se generan corriendo `prompts/04-simular-guiones.md`.

Durante la rutina semanal, cada guion publicado recibe una nota de
acierto/fallo al final de su archivo, comparando el rendimiento real
contra lo que la simulación estimó.
